
Pour ce projet # 2

On va utiliser l' API  OpenStreetMaps pour récupérer de l'information des différents restaurants ou bars selon le choix de l'utilisateur. et je les afficher dans un format de carte interactive (HTML). En plus, le programme fait du web scraping pour récupérer des avis des différents restaurants ou bars sur TripAdvisor.

Pour utiliser le programme sur le main on peut écrire la ville ou on veut faire la recherche et le type d’information qu’on veut récupérer (restaurants, bars, etc).
